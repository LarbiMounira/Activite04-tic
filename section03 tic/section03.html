Les Composants matériel d’un réseau informatique:
  1. Cartes réseau: Elle représente l’interface entre l’ordinateur et le réseau car elle permet la connection des appareils à internet et au réseau local. Les cartes d’interface réseau sont connectés souvent à une prise Ethernet ou Wi-Fi.
  2. Switch: Le commutateur est un équipement qui fonctionne comme un pont multiports et qui permet de relier plusieurs segments d'un réseau informatique local (LAN) entre eux. Il utilise les adresses MAC(Media Access Contrrol) pour communiquer les données uniquement à l’appareil destinataire.
  3. Hub: Le consentrateur est un appareil qui permet de concentrer les transmissions Ethernet de plusieurs équipements sur un même support dans un réseau informatique local (LAN).

Le Modèle OSI (Open Systems Interconnction):

Le modèle OSI peut être considéré comme un langage universel (référence) pour la mise en réseau d’ordinateurs.
La base du concept est de diviser un système de communication en sept couches abstraites, chacune assure un rôle particulier et communique avec les couches au-dessus et en dessous d'elle.
Les septs couches sont définies par :

  1. La couche physique: Elle inclut les équipements physiques impliqués dans le transfert de données.
  2. La couche de liaison de données:elle facilite le transfert de données entre les périphériques sur le même réseau.
  3. La couche réseau: Elle facilite le transfert de données entre des réseaux différents (routage de données).
  4. La couche transport : Elle est responsable de la communication de bout en bout entre les appareils.
  5. La couche session :(gestion des sessions entre appareils)Elle garantit que la session reste ouverte suffisamment longtemps pour transférer toutes les données échangées, puis la fermer.
  6. La couche présentation: Cette couche est principalement responsable de la préparation des données afin qu’elles puissent être utilisée par la couche applicative.
  7. La couche application:Elle interagit directement avec les applications sur les appareils de réseau .
    ** Le modèle OSI est un modèle théorique .Il représente les principes de communication réseau mais il est remplacé par le modèle TCP/IP dans le temps présent.

Le Modèle TCP/IP:
  • La couche de liaison de données.
  • La couche réseau.
  • La couche de transport.
  • La couche application.

L'adressage:
En général la communication se base sur :
  • Adressage
  • Architecture
  • Protocole
Prenant un exemple d’adressage, l’adressage IP :
Définition: est l’identificateur unique de l’appareil dans un même réseau; on distingue 2 types IPV4, IPV6.

IPV4 (internet protocole version4):

Les adresse sont forme par 32bits (4octets), cette suite de bits est compose de deux parties : Parties réseau et Parties hôtes
L’adressage IPV4 s’effectue par 4 types d'adressage:
  1. Adressage réseau: Les bits de sa partie hôte égalent a des 0.
    elle permi de savoi la possibilite de la communication entre 2 hôtes "machines",car toute machine dans un même réseau a la même partie
  2. Adressage de diffusion: Les bits de sa partie hôte égalent a des 1.
    elle est responsable de la diffusion de l'information a toutes les hôtes de ce réseau,a la fois.
  3. Adressage hôtes: c'est l'adresse IP unique de chaque machines dans un seul réseau.
  4. Masque réseau et sous-réseau : c'est le séparateur utilisé pour indique la partie hôte et réseau.

IPV6 (internet protocole version6):

Les adresse sont forme par 128bits (4octets), elle est decompose en 8 mots chacun de 16 bits,représente en hexadeciaml.
